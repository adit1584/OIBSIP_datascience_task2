# Unemployment Rate Analysis Using Data Science

## Description
Unemployment is measured as the percentage of unemployed people in the total labor force. During the Covid-19 pandemic, unemployment rates increased sharply across many regions. This project analyzes unemployment rate data using Python and Data Science techniques to understand trends, patterns, and the overall impact of Covid-19.

## Objective
To analyze unemployment rate trends before and during Covid-19 using data analysis and visualization techniques.

## Dataset
The dataset contains unemployment rate statistics including:
- Region/State
- Date
- Unemployment Rate
- Labor Participation Metrics

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
